Bintang keren

1. Descendant Selector (element spasi element) memilih element dari element yang di pilih. 2. Child Selector (element > element) memilih anak dari element yang di pilih. 3. Adjacent Sibling Selector (element + element) memilih element setelah element yang dipilih (sejajar dengan element nya). 4. General Sibling Selector (element ~ element) memilih seluruh element saudara dari element yang dipilih.
